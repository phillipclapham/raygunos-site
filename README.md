## What This Is

Standalone website for the RAYGUN OS cognitive architecture framework. A complete cognitive operating system for experiment-driven minds, built from 45 years of experiments and now open-sourced.

**Strategic purpose:** Separate universal methodology (viral spread to millions) from Adaptive Human products (commercial infrastructure for thousands). Zero commercial friction, pure value delivery.

---

## Tech Stack

**Philosophy:** Zero complexity. Proven by adapthuman.com.

- **Frontend:** Vanilla HTML/CSS/JS (no frameworks, no build process)
- **Fonts:** Syne (headings), Plus Jakarta Sans (body), Source Code Pro (code)
- **Deployment:** Cloudflare Pages (automatic deploys on push)
- **Design System:** Orange-led mad scientist palette, Lab Field background concept
- **Performance:** Fast, maintainable, works anywhere

## Design Decisions

**Color Palette:** Orange-Led Mad Scientist
- Primary: Orange/Burnt Orange (#E67E22 light, #F39C12 dark)
- Accent 1: Gold (#C9A961 light, #D4AF37 dark)
- Accent 2: Magenta (#db2777 light, #ec4899 dark)
- Ecosystem nod: Teal (sparing use in footer, subtle links)

**Signature Elements:**
- Lab Field background (3-layer: energy substrate, drifting orbs, micro bursts)
- Raygun laser divider (orange/gold beam, brain activation)
- Experiment orbs (spawn on clicks, orange/gold/magenta)
- Word glow effects (key phrases shimmer)
